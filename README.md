# FileMaker Probe
Crack passwords from FileMaker Pro databases versions 3.0 to 5.5. More discussion at <a href="https://macintoshgarden.org/apps/filemaker-probe">Macintosh Garden</a>.

# Building
1. Decompress **FileMakerPro.mcp.sit** and **FileMakerPro.rsrc.sit** (compressed with DropStuff 5.5).
2. Drop in all other source files into the same folder.
3. Compile with CodeWarrior 6.
